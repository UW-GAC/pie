Phenotype Inventory Explorer

PIE allows TOPMed investigators to view and search through the phenotype data currently available to the TOPMed Data Coordinating Center. The DCC is in the process of combining source phenotypes across TOPMed projects to produce harmonized phenotypes. TOPMed working groups can help the harmonization process by submitting details for how to combine source phenotypes using the harmonization request form.
