Help with phenotype tagging

How to tag phenotypes

From the "Tag phenotypes" form

  • In the navbar, select "Tag phenotypes" from the "Phenotype tags" menu.
  • In the "Tag" field, you may either select a tag from the dropdown menu or start typing to autocomplete the tag by name. When autocompleting, hit return to select the tag of your choice.
  • In the "Phenotype(s)" field, start typing the non-zero digits of the dbGaP variable accesssion (phv) for the phenotype variable you wish to tag. Hit return to select the phenotype variable of your choice.

    e.g. for phv00000803.v1.p10 (variable "MF353" from Framingham Heart Study), start typing "803"

  • You may select multiple dbGaP variable for the "Phenotype(s)" field. All of the selected variables will be labeled with the selected tag.
  • Click on the "Save" button to label the selected variable(s) with the selected tag.

From a tag's detail page

  • On a tag's detail page, click the 'Tag phenotypes as "[tag name]"' button. You will be taken to a form to select dbGaP phenotype variables to tag.
  • In the "Phenotype(s)" field, start typing the non-zero digits of the dbGaP variable accesssion (phv) for the phenotype variable you wish to tag. Hit return to select the phenotype variable of your choice.

    e.g. for phv00000803.v1.p10 (variable "MF353" from Framingham Heart Study), start typing "803"

  • You may select multiple dbGaP variable for the "Phenotype(s)" field. All of the selected variables will be labeled with the selected tag.
  • Click on the "Save" button to label the selected variable(s) with the specified tag.

From a study phenotype's detail page

  • On a study phenotype's detail page, click the "Tag this phenotype" button. You will be taken to a form to select the tag with which to label this phenotype.
  • In the "Tag" field, you may either select a tag from the dropdown menu or start typing to autocomplete the tag by name. When autocompleting, hit return to select the tag of your choice.
  • Click on the "Save" button to label the specified variable with the selected tag.

Terminology

Tag

A tag is a label representing a phenotype concept. This is usually synonymous with a phenotype name. Click on a tag's name to see its detail page, where you can read the detailed description of the phenotype concept it represents and specific instructions for identifying dbGaP variables to apply the tag to.

Tagged phenotype

A tagged phenotype is the association between a phenotype tag and a specific dbGaP variable (study phenotype).
